
# Clocket (WIP)

A Full Stack Pomodoro

## Run Locally

Clone the project

```bash
  git clone https://github.com/andasan/clocket.git
```

Go to the project directory

```bash
  cd clocket
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
    
## Tech Stack

**Client:** NextJs, daisyui/Twind

**Server:** HarperDB
